Single HTML file with a zoomable data canvas containing rectangles.

Features:

* Move, resize, rename, re-parent, and delete rectangles.
* Grid.
* Supports an extremely deep hierarchy, with a maximum depth approaching `Number.MAX_SAFE_INTEGER`.
* Save and load data locally or from files.

Data files are encrypted with a password using a simple XOR-CBC encryption algorithm.

Like circles more? Check this repository:
https://github.com/OneTimePads/BubbleDataCanvas
